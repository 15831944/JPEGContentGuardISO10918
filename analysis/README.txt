Overview:
----------
The analysis directory contains various scripts for generating analysis information
for tools used for analyzing the libjpeg library.  Lower level scripts may be ran
individually for specific types of data.  A top level analyze.py script calls all
the lower level scripts.


Running:
----------
To run the script:
   cd <analysis directory>
   ./analyze.py


General Notes:
----------
The analyze.py script must be ran within the analysis directory.
