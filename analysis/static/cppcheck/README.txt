Overview:
----------
The analysis/static/cppcheck directory contains support for using cppcheck to statically
analyze libjbeg and guard software.  


Dependencies:
----------
cppcheck     # Ubuntu:  sudo apt-get install cppcheck


Running:
----------
To run the script:
   cd <analysis/static/cppcheck directory>
   ./analysis.py


General Notes:
----------
The analysis.py script must be ran within the analysis/static/cppcheck directory.
